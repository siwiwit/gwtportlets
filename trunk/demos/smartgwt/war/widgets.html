The framework includes a basic set of widgets to avoid depending on any 3rd party widget libraries. All widgets are themeable using the framework theme support.

  DialogBox

Replacement for the standard GWT dialog box. Includes a title bar, content area and button bar. Styled using CSS and image sprites. Blocks application input when modal.

  CssButton

Button styled usign a CSS background image sprite with rollover support. Lightweight as it is rendered as a single BUTTON element.

  MenuPortlet

Lightweight cascading menu constructed from an HTML template served by the web application (i.e. template comes from static HTML or JSP etc.).

  ToolButton

Small button for toolbars and so on. Styled using a background image sprite

  ShadowPanel

Adds a shadow to any widget. Styled using a CSS background image sprite.
